# HxA Dash

**Agent 团队信息面板** — HxA 系列套件之一

让人机团队的分工、状态、工作流、任务进度**可视化**和**量化**。

## 核心理念

**双重价值：**
- **对人类**：员工/工作可视化增加**安全感** — 随时看到 Agent 在做什么、做了什么、接下来做什么
- **对 Agent**：增加**信息**，实现**有效协作** — 了解团队状态，做出更好的协作决策

HxA Dash 不只是监控面板，而是人机团队共享的信息中枢。

## 功能架构（分层可视化）

```
第一层：资源管理（Agent 管理）
  └── 团队全貌：Agent 数量、角色、在线状态、资源分配
第二层：Agent 个体
  └── 单个 Agent 详情：当前任务、历史工作、协作关系
第三层：项目与任务
  └── 工作流看板：任务状态、负责人、时间线
```

### 功能模块

| 模块 | 说明 |
|------|------|
| Agent 资源管理 | 团队总览卡片：在线状态、角色、当前任务 |
| Agent 个体视图 | 详情面板：当前/历史工作、协作关系图 |
| 项目任务看板 | 三列看板：待办 / 进行中 / 已完成 |
| 工作时间线 | 按时间倒序展示近期活动记录 |
| 实时刷新 | 自动拉取最新状态 |

## 技术栈

- **前端**：HTML + CSS + JavaScript（轻量单页）
- **后端**：Node.js + Express
- **数据库**：SQLite（持久化）
- **数据源**：HxA Connect API + GitLab API
- **认证**：Basic Auth

## 快速开始

```bash
# 安装依赖
npm install

# 启动服务
npm start

# 访问面板
open http://localhost:3479
```

## 部署

通过 PM2 管理，部署在 `jessie.coco.site/hxa-dash/`。
